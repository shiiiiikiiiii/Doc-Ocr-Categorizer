# react+antd制作管理系统

[服务器端接口](https://github.com/btc022003/hdf-app-server)

## 初始化项目

```bash
npm init vite@latest # 根据提示一步一步就行
cd 你的项目
npm i
```

## 使用到的插件

安装以下插件

nprogress是一个进度条插件

当前时间2022-11

```bash
npm i react-router-dom axios nprogress antd @ant-design/icons
```

## 启动项目

```bash
npm run dev # 启动
```
